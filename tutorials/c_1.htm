Using the distance transform operation


    Step-by-step
 

 

In order to attain this purpose the following steps need to be followed:

Click on Preprocess –> Scence Operations -> Filter.
(Files brorwing window will pop up)

Choice the the image file (BIM) which need for correction.
For example: knee.BIM

At Down-Right corner, on Filter : Type, Click and Choice on Dist2Dxy or Dist3D.

Now you can clik on Save to make a new IMO file.

The new IM0 file is the distance transformed image file.
