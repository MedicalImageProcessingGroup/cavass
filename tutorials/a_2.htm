Importing DICOM data

DICOM data is the format which data is obtained from the scaner. However, in order to work on the data it needs to be converted into native CAVASS data format, (IM0 file). Follwing procedures will teach you how to import your DICOM files into CAVASS.

 
    Step-by-step

     

     

     
To start a new project, do one of the following:
  • Choose File > Import > DICOM

  • Then Click on Input, on Lower right coner.

  • The name of the selected DICOMs will be displayed. Click Open. It also displays the number of slices the output IM0 will have
  • Next, type the name of the output file (without extensions) and the place, Then click on Save
