How to import and export data
 

In order to access the image file, The application need to setup the input and output path so the data could be uitliz
    Step-by-step
 
To begin the application
  • Click on the Edit button located on the upper tools command. Choice the Preferences..

To set to the paths:
  • Click on the Directories tap located on the Preferences windows. Key in the directoried information for the CAVASS home directory, Input directory and Output directory

  • Then click OK to get out of this window. Now your CAVASS software are ready to access the data.
