How to correct the non-uniformity present in an image

The Image sensor has various measurable imperfections, such as geometric distortions due to main field variations, radio-frequency inhomogeneities causing "image shading", and imperfect slice profiles.
    Step-by-step


In order to attain this purpose the following steps need to be followed:

Click on Preprocess –> Scence Operations -> Filter.
(Files brorwing window will pop up)

Choice the the image file (IM0) which need for correction.
For example: knee.IM0

At Down-Right corner, on Filter : Type, Click and Choice on Inhomo.

Now you can clik on Save to make a new IMO file.

The new IM0 file is the inhomogeneities corrected image file.
