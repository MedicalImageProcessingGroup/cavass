Interpolation: change spacing and number of slices

 
    Step-by-step

 

In order to attain this purpose the following steps need to be followed:

First, make sure that your input file is been selected.
By click File -> Input..
In browsing window: high lite your file and click OK.

Click on Preprocess –> Scence Operations -> Interpolate

The screen will show up your input image slices, with slices number.
For example: knee.IM0

In this steps:

We could change X3 (Z axis) value:
Defult is XY pixal's value
For method: We could click between NearestNeighbor, Linear, Cubi

After all above, we could click Save. This will generate a new image file.
