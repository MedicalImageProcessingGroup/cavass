Create a new IM0 from an existing IM0

We imported the IM0 file, and we did some image processing.
Now we could export the Region of Interest (ROI) to a new IM0 file for more Radiology Task.
    Step-by-step
 
To create a new IM0 file, do one of the following:
  • Choose File > Save. (If Save is inactive, the project is a type that cannot be saved.)
  • Click on the Save button in the lower left corner of the project window.
To open a saved project, do one of the following:
  • Choose File > Open. Use the window to navigate to the file. Select the file and click on the Open button.
  • From the File menu, select a project from the list of recently saved projects.
  • Using Windows Explorer, navigate to the saved project file. Double-click on the file name, and the project will open.
