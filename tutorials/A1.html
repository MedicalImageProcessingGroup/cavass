A.1 Learn the user interface

The major file types encountered in CAVASS are as follows:

  • IM0 - multidimensional image data
  • BIM - multidimensional binary images
  • BS0, BS1, BS2 - surfaces represented by voxels, voxel faces, triangles, respectively
  • SH0 - fuzzy boundaries represented as shells used in volume rendering
  • MV0 - animation or movie files

Select an input file, say regular IM0 by following the steps given below.

  • Click on main menu File and select New
    • Use the Browse button to find and select the file regular IM0
  • Left click on Visualize and then select Slice > Cycle
    • Now in the Image Window, a set of slices are displayed. In the Dialog Window (in the lower portion of the screen), at the right corner, a panel of options is displayed. Select “magnify” under Mode from the drop-down list. Note at the bottom right corner of the screen function indicators for the three mouse buttons. They read Position/Size, OK, and cancel.
  • Left click on any slice displayed in the image window. This slice is selected now and can be magnified and positioned anywhere in the image window using the Position/Size options. Click middle button (OK) to accept and deposit this slice display.
  • Left click on Cine in panel. A CineOptions window will appear on the bottom left. The speed of cine display of slices can be controlled by the value of “delay”. By left clicking on “forward” or “forward/backward”, the cine sequence can be displayed and controlled.

The user interface style exemplified in this Visualize-Slice-Cycle command is used throughout CAVASS: Images and graphics are displayed in the Image Window. The Panel Window appears on the lower right and allows choosing various options associated with the current command. Further widgets appear on the left part of the Dialog Window. The mouse functions are always shown at the very bottom of the screen on the right corner.
