Read and alter header information of the input file

We have the image files (IM0, BIM, BS2..) in input file. We can get the information of the images. Also we can modify those header information if we need.
    Step-by-step
 
To read header informationof the input files, do the following:
  • Click on File -> Import -> EasyHeader.
  • In the upper screen: Show image's information
    Those information include:
    Dimension, Width, Height, Bits, Pixel size, Min and Max Density...
To change header informationof the input files, do the following:
  • Click on File -> Import -> EasyHeader.
  • In the lower screen: We can change image's information in the box.
  • Click on SAVE NEWHDR to save the new information in header file.
